Baukasten für Copilot-Chat-Agenten

Dieser Baukasten unterstützt dich dabei, einen vollständig einsatzbereiten Copilot Chat Agenten zu erstellen. Du wählst die gewünschten Eigenschaften aus, passt die erzeugten Textbausteine bei Bedarf an und kopierst anschließend das fertige Agenten Script in einen neuen Agent in Copilot.

Viel Erfolg beim Erstellen deines Agents.

Datenschutzhinweis

Der Baukasten speichert keine persönlichen Daten. Alle Eingaben bleiben ausschließlich in deinem Browser und werden nicht an Server übertragen. Beim Schließen der Seite gehen nicht gespeicherte Inhalte verloren.

Dieser Baukasten ist in logische Module unterteilt, die ein korrekt aufgebauter Agent benötigt. In allen Modulen außer dem ersten kannst du die Texte in den grünen Feldern nach Bedarf ändern. Bitte achte darauf, dass die inhaltliche Richtung und die maximale Auswahl erhalten bleibt. Unbearbeitete Module fließen nicht in das Agenten Script ein. Sobald alle Module ausgefüllt sind, kannst du das fertige Agenten Script mit einem Klick in die Zwischenablage kopieren.

Anleitung

  1. Neuen Agenten anlegen
    Öffne in Copilot Chat „Neuer Agent”. Dieses befindet sich auf der linken Seite.
  2. Bezeichnung eintragen
    Gib deinem Agenten einen kurzen, klaren Namen. Beispiele: Office Assistent, Begleiter, Coach, Azubi Tutor.
  3. Beschreibung eintragen
    Beschreibe in ein bis zwei Sätzen, wofür der Agent gedacht ist. Beispiel: „Unterstützt bei täglichen Aufgaben, erklärt Inhalte verständlich und liefert klare, zuverlässige Antworten.“
  4. Agent-Script einfügen
    Füge den kopierten Text aus dem Baukasten in das Feld „Anweisungen“ ein. Das Script bildet die komplette Funktionslogik des Agents.
  5. Agent speichern und verwenden
    Klicke auf Erstellen, um deinen neuen Agenten anzulegen.
  6. Bearbeiten des Agent
    Wenn die Ergebnisse des Agent nicht zufriedenstellend sein sollten, ist es möglich das Agent-Script aus den Anweisungen zu kopieren und in einem regulären Copilot Chat anzupassen. Klicke dazu neben dem Agent auf die drei Punkte und wähle „Bearbeiten“.
