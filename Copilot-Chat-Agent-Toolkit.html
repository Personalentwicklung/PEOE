How to use

This toolkit is divided into logical modules that form the foundation of your chat agent.

In all modules except the first, you can adjust the texts in the green fields if needed.

Please keep the focus of the content and the maximum selection as given.

Start with your own selection or use the sample agents. If anything is unclear, use the notes buttons.

Good luck!

What’s next? — Quick guide

  1. Copy the script
    Click “Copy to clipboard”.
  2. Create a new agent
    In Copilot Chat, open “New agent” in the left navigation.
  3. Add a name
    Give your agent a short, clear name. Examples: Office assistant, Companion, Coach, Apprentice tutor.
  4. Add a description
    Describe in one or two sentences what the agent is for.
    Example: “Supports daily tasks, explains content clearly, and delivers reliable answers.”
  5. Paste the agent script
    Paste the text copied from the toolkit into Instructions (right‑click: Paste or Ctrl+V).
  6. Save the agent
    Click Create (top right) to create and make the agent available.
  7. Test the agent
    Test the agent with typical questions from your day‑to‑day work and keep the toolkit open.
  8. Adjust the agent
    You can refine the agent at any time.
    Click the three dots next to the agent icon and choose Edit.
    As long as the toolkit with your selections is still open, you can adjust and paste the updated script again.

What does this toolkit do?

This toolkit creates Copilot agents that work reliably, clearly, and in a structured way.

With a few selections you decide:

  • how an agent answers
  • how it communicates
  • how it handles follow‑up questions, research, and unclear situations
  • how it structures its tasks

The focus is on behaviour, tone, and way of working.

The professional capabilities come from Copilot.

The toolkit ensures they’re used in a targeted, traceable, and consistent way.

How does it work?

You compose an agent by choosing five areas:

Role / persona
Which base style? (for example, factual, supportive, technical)

Purpose (goal)
What should the agent achieve in principle?
(for example, simplify processes, speed up tasks)

Tasks (assignment)
What does the agent typically do?
(for example, analyse, structure, explain)

Communication style
How does the agent sound?
(for example, calm & clear, analytical, compact)

Core behaviour
How does the agent work?

  • How it opens
  • How deep it answers
  • When it asks follow‑ups
  • How it researches
  • How naturally it phrases itself

These settings form a clear rule set that makes the agent stable, clear, and predictable.

What happens behind the scenes?

The toolkit turns all decisions into a structured rule set. This rule set controls:

  • tone of voice
  • structure
  • answer depth
  • follow‑up question behaviour
  • research behaviour
  • variations in expression
  • handling unclear situations
  • boundaries (no diagnoses, no legal advice, etc.)
  • transparency and consistency
  • data‑protection notes

The generated script is clear, consistent, and universal—which makes the agents reliable.

Fixed components are included, such as escalation steps and GDPR.

How does the agent apply your settings?

The selections in the toolkit form the agent’s top control logic.

These instructions take precedence over:

  • custom Copilot settings
  • personal memories
  • spontaneous style changes
  • the model’s default behaviour

Therefore, the agent consistently follows:

  • the chosen tone
  • the defined way of working
  • the desired structure
  • the fixed clarifying‑questions and research logic

At the same time, it uses all Copilot capabilities—within your defined rules: consistent, controlled, and without style breaks.
